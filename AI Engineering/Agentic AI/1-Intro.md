## Quick Revision Cheatsheet

| Concept | One-liner |
|---|---|
| **Agentic AI** | AI that autonomously plans & acts across multiple steps (ReAct loop) |
| **Self-Modifying Prompt** | AI rewrites its own system prompt to accumulate knowledge across sessions |
| **Prompt Contract** | Hard rules embedded in the system prompt for consistent AI behavior |
| **Reverse Prompting** | AI asks clarifying questions before acting on ambiguous tasks |
| **Context Compression** | Auto-summarizing old context when nearing the token limit |
| **MCP Orchestration** | Multiple specialized agents connected via MCP, coordinated by an orchestrator |
| **Router Decision Hub** | Agent/logic that routes tasks to the right specialist agent |
| **Stochastic Consensus** | Run same query N times → aggregate outputs for more reliable answers |
| **Agent Chat Rooms** | Shared message space where agents self-coordinate without rigid chaining |
| **Debate among Agents** | Agents argue opposing views; a judge picks the best-reasoned answer |
